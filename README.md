# IISER Mohali Robotics Club - Club Room Rulebook

This repository contains the official rulebook for the IISER Mohali Robotics Club Room, defining safe and fair use of the club facilities for projects, storage, fabrication, and testing.

## About

The rulebook applies to all members, guests, and collaborators of the IISER Mohali Robotics Club. It establishes guidelines for proper usage of the club room and its resources.

## Repository Contents

- `rulebook.tex` - Main LaTeX source file for the rulebook
- `rulebook.pdf` - Generated PDF (after compilation)

## Compilation Instructions

**Important:** This document must be compiled using **XeLaTeX**.

> NOTE: Inital writer of the rulebook used Tex Live(8 March 2025 release) distribution on Windows with XeLaTeX engine.

### Using LaTeX Editors

- **TeXstudio/TeXmaker:** Set XeLaTeX as the default compiler in preferences/settings
- **Overleaf:** Select "XeLaTeX" from the compiler dropdown menu in the project settings
- **VS Code (with LaTeX Workshop):** Configure the build recipe to use XeLaTeX
